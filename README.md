# Zubal Clothing

A modern e-commerce web application built with Flask for managing and selling clothing products online.

---

## Features

- Customer Registration & Login
- Product Catalog
- Product Categories
- Featured Products
- Product Reviews
- Shopping Cart
- Wishlist
- Checkout
- WhatsApp Order Integration
- Related Products
- Stock Management
- Order Management
- Admin Dashboard
- Product Image Upload
- Responsive Design
- Professional UI/UX

---

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript
- Jinja2

---

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
